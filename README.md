# 🏛️ نظام إدارة المعهد العلمي (Institute Management System)

تطبيق ويب تقدمي (PWA) مصمم خصيصاً لإدارة شؤون معهد علمي يعتمد نظام التدريس في عطلة نهاية الأسبوع (السبت والأحد). يهدف النظام إلى رقمنة العملية التعليمية وتسهيل التواصل بين الإدارة، الأساتذة، والطلبة.

## ✨ المميزات الرئيسية

ينقسم النظام إلى ثلاث واجهات رئيسية بصلاحيات مختلفة:

1. **👑 لوحة المدير العام (Admin Panel):**
   - إدارة الهيكلة (الأفواج، الأقسام، المواد الدراسية).
   - إضافة وحذف المستخدمين (أساتذة وطلبة).
   - نشر الإخبارات والمواعيد العامة.
   - إحصائيات عامة عن المعهد.

2. **👨‍🏫 بوابة الأستاذ (Teacher Portal):**
   - تسجيل الحضور والغياب اليومي للفوج.
   - رفع الملخصات والدروس (PDF, Word).
   - تكليف الطلبة بواجبات وتقييم إنجازاتهم (التنقيط).
   - إرسال إشعارات مخصصة لطلبة الفوج.

3. **🎓 بوابة الطالب (Student Portal):**
   - متابعة نسبة الحضور وأيام الغياب.
   - تصفح وتحميل الدروس والملخصات من المكتبة.
   - متابعة الإشعارات العامة والخاصة بالمواد.
   - الاطلاع على الواجبات المطلوبة وإرفاق الإجابات.

## 🛠️ التقنيات المستخدمة (Tech Stack)

تم بناء هذا النموذج الأولي (Prototype) ليكون خفيفاً وسريعاً باستخدام تقنيات الواجهة الأمامية:
- **HTML5:** لبناء الهيكل الأساسي للواجهات.
- **Tailwind CSS:** لتصميم واجهات عصرية ومتجاوبة مع كافة الأجهزة.
- **Vanilla JavaScript:** للتعامل مع المنطق البسيط (مثل التحقق من الدخول والتبديل بين التبويبات).
- **PWA (Progressive Web App):** تم تضمين ملفات `manifest.json` و `Service Worker` لتمكين تثبيت الموقع كتطبيق على الهواتف الذكية والحواسيب.

## 🚀 طريقة الرفع والاستضافة مجاناً (GitHub Pages)

لجعل هذا التطبيق متاحاً للجميع عبر رابط مباشر، اتبع الخطوات التالية:

1. قم بإنشاء مستودع جديد (New Repository) على حسابك في GitHub.
2. ارفع جميع ملفات المشروع (المجلدات والملفات بما فيها `index.html`) إلى المستودع.
3. اذهب إلى إعدادات المستودع **Settings**.
4. من القائمة الجانبية، اختر **Pages**.
5. تحت قسم *Build and deployment*، اختر الفرع الرئيسي (غالباً `main` أو `master`) ثم اضغط **Save**.
6. انتظر دقيقة أو دقيقتين، وسيظهر لك رابط التطبيق المباشر لترسله للأساتذة والطلبة!
