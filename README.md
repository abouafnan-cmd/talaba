# نظام الحضور والغياب

تطبيق ويب متكامل لإدارة حضور وغياب الطلاب في المؤسسات التعليمية.

## المميزات

- ✅ تسجيل حضور وغياب الطلاب
- 👥 إدارة الطلاب (إضافة، تعديل، حذف، استيراد)
- 📚 إدارة المواد الدراسية والأقسام
- 📊 تقارير وإحصائيات متنوعة
- 📄 تصدير التقارير إلى PDF
- 📱 تصميم متجاوب يعمل على جميع الأجهزة
- 💾 حفظ البيانات محلياً في المتصفح

## التقنيات المستخدمة

- HTML5
- CSS3 (Grid, Flexbox, Custom Properties)
- JavaScript (ES6+)
- Local Storage لتخزين البيانات
